## 📂 폴더 구조

```
JSP/
├── 📁 servletstudy/         # Servlet 기초
├── 📁 jspstudy/             # JavaServer Pages 기초
├── 📁 model1study/          # MVC Model 1 패턴
├── 📁 model2study/          # MVC Model 2 패턴
├── 📁 mybatisstudy/         # MyBatis ORM 프레임워크
├── 📁 jquerystudy/          # jQuery 기초 및 DOM 조작
├── 📁 ajaxstudy/            # Asynchronous JavaScript And XML
├── 📁 jsoupstudy/           # jsoup HTML 파싱
└── 📁 guestbook/            # MVC Model 2 실전 프로젝트
```

## 📖 커리큘럼

| 단계 | 폴더 | 주제 | 핵심 개념 |
|:----:|------|------|-----------|
| 1 | `servletstudy` | Servlet | HttpServlet, doGet/doPost, request/response |
| 2 | `jspstudy` | JavaServer Pages | EL, JSTL, include, forward |
| 3 | `model1study` | MVC Model 1 | JSP에서 직접 DB 연동 |
| 4 | `model2study` | MVC Model 2 | Controller / View 분리 패턴 |
| 5 | `mybatisstudy` | MyBatis | SqlSession, Mapper XML, CRUD |
| 6 | `jquerystudy` | jQuery | 선택자, 이벤트, DOM 조작 |
| 7 | `ajaxstudy` | AJAX | XMLHttpRequest, fetch, 비동기 처리 |
| 8 | `jsoupstudy` | jsoup | HTML 파싱, 웹 크롤링 |
| 🏁 | `guestbook` | 실전 프로젝트 | MVC Model2 기반 방명록 CRUD |
